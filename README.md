# Blazor Web Project

## Overzicht
Een Blazor-webapplicatie met als doel de Ingress Community van informatie rondom First Saturday events te voorzien. 
De applicatie wordt ontwikkeld voor daadwerkelijk gebruik binnen de Ingress-community
en groeit mee met de behoeften rondom First Saturday events.

## Doel van het project
Dit project is opgezet om ervaring op te doen met Blazor, component-based architectuur
en het opzetten van een overzichtelijke front-end structuur binnen .NET.

## Functionaliteit
- Component-gebaseerde pagina-opbouw
- Herbruikbare UI-componenten
- Responsive layout voor desktop en mobiel

## Technische keuzes
- Blazor
- Component-scoped CSS
- Focus op herbruikbaarheid en duidelijke naamgeving

## Status
In ontwikkeling. Functionaliteit en styling worden stapsgewijs uitgebreid.

## Opmerking
De focus ligt op structuur, componentontwerp en onderhoudbaarheid,
niet op visuele perfectie.
